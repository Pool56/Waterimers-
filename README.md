# Waterimers-
This is an autonomous lead qualification agent

## Use Case

A fully automated AI sales assistant that:

Captures leads from website forms

Researches the company using Perplexity API

Classifies lead quality using Claude Sonnet

Routes leads to sales team automatically

Updates CRM in Airtable

Sends Slack notifications


## Tech Stack

# N8N

# Claude Sonnet

# Perplexity API

# Airtable

# Slack API

# Webhooks

## Flow

1. Website Form
     
2. Webhook Trigger
     
3. Perplexity Company Research
     
4. Claude Lead Scoring
     
5. CRM Update (Airtable)
     
6. Slack Notification
